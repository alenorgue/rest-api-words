REST API Words

Esta API te permite obtener palabras aleatorias.

Endpoints v1

  • GET /api/v1/words
    Devuelve una palabra aleatoria.
  • GET /api/v1/words?length=3
    Devuelve una palabra aleatoria de 3 letras (entre 3 y 13).
  • GET /api/v1/words/all
    Devuelve todas las palabras disponibles.
  • GET /api/v1/words/:index
    Devuelve la palabra en la posición :index del array de palabras. Si el índice no existe, devuelve un error 404.

Ejemplo de respuesta v1

{
  "word": "mar"
}

Endpoints v2

  • GET /api/v2/languages
    Devuelve la lista de idiomas disponibles.
  • GET /api/v2/words
    Devuelve una palabra aleatoria.
  • GET /api/v2/words?length=5&lang=it
    Devuelve una palabra aleatoria de 3 letras (entre 3 y 13) y en el idioma seleccionado (de la lista de idiomas disponibles).

Ejemplo de respuesta v2

{
  "word": "cuore",}
